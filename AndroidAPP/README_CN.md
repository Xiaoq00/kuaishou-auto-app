# 快手极速版自动刷视频APP

基于Android无障碍服务的自动刷视频应用，支持随机滑动、反检测等功能。

## 🚀 在线构建说明

### 方法一：GitHub Actions（推荐）

1. **Fork项目到你的GitHub账户**
   - 点击GitHub页面右上角的"Fork"按钮
   - 将项目复制到你的账户下

2. **启用GitHub Actions**
   - 进入你的仓库页面
   - 点击"Actions"标签
   - 点击"I understand my workflows, go ahead and enable them"

3. **触发构建**
   - 推送代码到main分支，或手动触发Actions
   - 进入"Actions"标签查看构建进度

4. **下载APK**
   - 构建完成后，在"Actions"页面找到对应的构建记录
   - 点击进入详情，在"Artifacts"部分下载APK文件

### 方法二：本地构建

1. **安装Android Studio**
2. **导入项目**
3. **构建APK**：Build → Build APK(s)

## 📱 功能特性

- ✅ 无障碍服务自动刷视频
- ✅ 开始/暂停/停止控制
- ✅ 30分钟时间限制
- ✅ 随机滑动间隔（8-22秒）
- ✅ 反检测自然滑动
- ✅ 悬浮窗状态显示

## 🔧 技术规格

- **目标平台**：Android 8.0+ (API 26+)
- **开发语言**：Kotlin
- **核心组件**：无障碍服务(AccessibilityService)

## 📋 使用说明

1. **安装APK**到Android手机
2. **开启无障碍权限**：
   - 设置 → 无障碍 → 已下载的服务
   - 找到"快手自动刷视频"并开启
3. **启动应用**，点击"开始脚本"
4. **应用会自动打开快手极速版并开始刷视频**

## ⚠️ 注意事项

- 确保手机已安装快手极速版
- 首次使用需要授权无障碍权限
- 建议在WiFi环境下使用
- 请勿用于商业用途
